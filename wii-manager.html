Scripts de integración

Esta GUI genera y ejecuta comandos wit / wwt vía zenity o directamente en bash. Para integrarlo con tu escritorio, usa el lanzador incluido.

Referencia de comandos wit / wwt
wwt LIST -p /ruta/wbfs — listar juegos en WBFS
wwt ADD -p /ruta/wbfs juego.iso — añadir ISO al WBFS
wwt EXTRACT -p /ruta/wbfs ID --dest /dst/ — extraer a ISO
wwt REMOVE -p /ruta/wbfs ID — eliminar juego
wwt VERIFY -p /ruta/wbfs — verificar todos los juegos
wit COPY src.iso dst.wbfs — copiar / convertir
wit VERIFY juego.iso — verificar integridad
wit LIST juego.iso — info del juego
wwt --auto LIST — detectar WBFS automáticamente

wwt ADD --wfs [dst] [src]
